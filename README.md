This repo is meant to contain useful scripts to use in LMI's GoPxL Python Script tool.\

Scripts must adhere to the following convensions

Header comments:\
#date created\
#author\
#number of inputs\
#input type 1\
#input type 2\
#etc.\
#tool description\
\
Footer comments:\
#number of outputs\
#output type 1\
#output type 2\
#etc.
